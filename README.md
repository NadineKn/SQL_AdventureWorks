# AdventureWorks - Försäljningsanalys

## Sammanfattning:
**Huvudsakliga fynd**
- Bikes är största produktkategorin, Mountain Bikes och Road Bikes är största subkategori.
- Generellt stigande trend över åren 2022 - 2025. Det finns toppar under sommaren och lägre försäljning under vintern.
- Regioner Southwest, Canada och Northwest genererar mest intäkter.
- Regioner Central, Northeast, Southeast har högst genomsnittligt ordervärde.

**Rekommendationer**
- Vad kan företaget göra baserat på denna analys?
- Fokusera på starka regioner med hög försäljning för att maximera intäkterna.
- Förbättra svaga regioner, analysera varför vissa regioner presterar sämre.
- Satsa på produktkategorier som säljer bäst i varje region.
- Överväg att minska fokus på lågt säljande kategorier i svaga regioner.
- Planera kampanjer och erbjudanden efter säsongsmönster.
